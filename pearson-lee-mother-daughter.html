Karl Pearson organized the collection of data on over 1100 families in England in the period 1893 to 1898. This particular data set gives the heights in inches of parents and their children (of the same sex), with up to two children per parent. All children are at least age 18, and all parents are younger than 65. Data were given in the source as a frequency table to the nearest inch.

Estimate $r$ and $r_s$¶

Calculate the residuals from the normal linear model equation

$$y = r x + c+ \epsilon$$
$$\epsilon \sim \mathcal{N}(0, r_s^2 \sigma^2)$$
$$\sigma^2 = 1$$
$$\hat{r}_s = \sqrt{RSS / (n-p)}$$
$$p = 2$$

# Manually calculate the beta coefficients
X = np.concatenate([[np.ones(x.size)], [x]]).T
Y = y.reshape(-1, 1)

beta = np.linalg.inv(X.T @ X) @ X.T @ Y
beta = beta.ravel()

# Test that the number of coefficients is two 
# (the intercept  and the coefficient)
p = 2
assert beta.size == p

# Test that the manual calculation agrees with 
# the SkLearn software calculation
assert np.allclose(beta, (c, r))

# Calculate the residual coefficient estimate (see the equation)
# The numerator is the sum of squares (dot product)
rs = np.sqrt(np.dot(resid, resid) / (n - p))

# Check that it's close to the standard deviaiton of the residuals
assert np.allclose(resid.std(), rs, atol=0.01)
